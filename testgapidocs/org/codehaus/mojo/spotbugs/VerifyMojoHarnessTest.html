    • org.codehaus.mojo.spotbugs.VerifyMojoHarnessTest 
    • @org.apache.maven.api.plugin.testing.MojoTest
class VerifyMojoHarnessTest
extends java.lang.Object

      Tests for org.codehaus.mojo.spotbugs.VerifyMojo using the Maven Plugin Testing Harness (JUnit 5).

      The verify goal reads a pre-existing SpotBugs XML file and fails the build if violations are found. It does not invoke SpotBugs itself.
