Package: org.codehaus.mojo.spotbugs

[Groovy] Class SpotBugsMojo

      • org.codehaus.mojo.spotbugs.SpotBugsMojo
      • All Implemented Interfaces and Traits:
        SpotBugsPluginsTrait
        @org.apache.maven.plugins.annotations.Mojo(name: spotbugs, requiresDependencyResolution: ResolutionScope.TEST, requiresProject: true, threadSafe: true)
class SpotBugsMojo
extends AbstractMavenReport
implements SpotBugsPluginsTrait

        Generates a SpotBugs Report when the site plugin is run. The HTML report is generated for site commands only.

            File names of the baseline files. Bugs found in the baseline files won't be reported.

            Potential values are a filesystem path, a URL, or a classpath resource.

            This parameter is resolved as resource, URL, then file. If successfully resolved, the contents of the configuration is copied into the ${project.build.directory} directory before being passed to Spotbugs as a filter file.
