Package: org.codehaus.mojo.spotbugs

[Groovy] Class SpotBugsAggregateMojo

      • org.codehaus.mojo.spotbugs.SpotBugsAggregateMojo 
      • @org.apache.maven.plugins.annotations.Mojo(name: spotbugs-aggregate, aggregator: true, requiresDependencyResolution: ResolutionScope.TEST, requiresProject: true, threadSafe: true)
class SpotBugsAggregateMojo
extends AbstractMavenReport

        Generates an aggregate SpotBugs report for multi-module projects when the site plugin is run. The HTML report is generated from the individual SpotBugs XML results of each module. Run spotbugs:spotbugs on each module before using this goal.

        Since:
        4.9.4.2
